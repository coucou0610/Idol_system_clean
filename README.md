# 🎤 偶像养成系统

> 适用于 SillyTavern 的偶像经纪公司管理扩展，为偶像养成类剧情提供完整的事务所运营界面。

> 🙏 此扩展是在 **Arnold1900** 老师制作的 CTE 男团经营扩展上进行修改，感谢 Arnold1900 老师的授权！

---

## 📦 安装方法

1. 打开 SillyTavern，进入「扩展程序」→「安装扩展」
2. 输入仓库地址：
   ```
   https://github.com/coucou0610/Idol_system_clean
   ```
3. 安装完成后刷新页面，点击扩展图标即可打开

---

## 🗂 功能模块

### 🗺 地图
- 显示京港市区域地图，包含主要地点标记
- 点击地点图标查看地点简介，可前往目的地触发剧情
- 支持切换国家地图，点击城市查看城市情报

### 📅 行程
- 自动从聊天记录中捕捉 `<status_bar>` 标签内的 `[Schedule|...]` 字段
- 显示当日安排时间表
- 支持读取并展示「已接通告」和「7日行程表」

### 📋 通告接洽
- 按类型筛选通告（影视 / 唱片 / 舞台 / 综艺 / 广告）
- 每张通告显示：项目名、公司、属性要求、酬劳、拍摄周期、截止日期
- 点击「签署合约」弹出确认窗口，显示通告详情与内容说明
- 确认后自动输出接取通告的文字到输入框
- 点击「生成通告」按钮按当前 API 设置生成新通告列表

### 🛒 采购部
- 按类型筛选商品（营销 / 团队 / 时尚 / 旅游 / 投资 / 设备等）
- 分类标签横向滚动，商品两列网格展示
- 点击「购买」后选择购买人与分配对象，输出采购文字
- 点击「刷新商品」按当前 API 设置生成新商品列表

### 🎵 HADES SOUND ATELIER
- **交互式创作引导**
  - 可选择创作角色、人声音域、声部音色、音乐流派、子流派、主乐器及歌词主题
  - 自动整理创作参数并生成音乐创作 Prompt
  - 调用偶像系统已保存的 API 设置，无需为音乐模块重复配置 API
- **智能创作笔记**
  - 捕捉并整理 LLM 返回的歌名、歌词与音乐风格
  - 支持查看生成结果、复制歌词和风格，并保留创作历史记录
- **音频播放器**
  - 支持上传本地 MP3 文件试听；本地文件仅在当前浏览器会话中有效
  - 支持添加 MP3 网络链接，并将链接保存至浏览器本地
  - 播放列表支持选曲、播放 / 暂停、上一曲、下一曲及删除歌曲
  - 显示当前歌曲名称、已播放时间和歌曲总时长
  - 提供可拖动播放进度条，可快速跳转到歌曲的指定位置

### 📰 京港娱乐日报
- 三栏报纸风格布局：
  - **左栏**：头条新闻（大标题 + 正文 + IMPACT 分析）
  - **中栏**：微博实时热搜排行榜 + 竞品动态
  - **右栏**：八卦爆料 + 行业资讯
- 顶部显示舆论风向、公关压力指数、今日热词
- 点击「生成日报」按钮按当前 API 设置生成当日新闻

## ⚙️ API 设置

点击左侧导航栏底部「⚙️」图标打开设置面板：

| 设置项 | 说明 |
|--------|------|
| API 模式 | 可选择「使用酒馆主 API」或「使用官方 Key」 |
| 使用酒馆主 API | 直接调用 SillyTavern 当前启用的主 API，无需单独填写地址和密钥 |
| 使用官方 Key | 使用本插件单独保存的官方 API 配置 |
| 官方 API 类型 | 可选 OpenAI、ChatGPT、Claude、Gemini、DeepSeek、MiniMax |
| API 地址 | 仅选择 OpenAI / OpenAI 兼容接口时需要填写 |
| API 密钥 | 填写所选官方接口对应密钥 |
| 模型 | 切换官方 API 类型后会自动填入默认模型，可点击「获取模型」读取可用模型，也可手动修改 |
| 温度 | 生成内容的随机性（建议 0.7~1.0） |
| Max Tokens | 最大生成长度（建议 3000 以上） |
| 上下文读取 | 默认读取最近 8 条聊天上下文 |

通告生成、商品刷新、日报生成与音乐创作都会读取这里的 API 模式设置：选择「酒馆主 API」时跟随 SillyTavern 当前连接；选择「官方 Key」时使用本面板保存的供应商、密钥和模型。OpenAI / OpenAI 兼容接口需要填写 API 地址，其余官方接口只需要填写密钥并确认模型。「获取模型」会尝试读取当前供应商的在线模型列表，若接口限制浏览器读取，则自动回退到内置推荐模型。

## 🔗 MVU 变量集成

扩展支持从 MVU（变量管理器）读取角色属性数据，路径为：

```
stat_data.玩家状态.属性.{歌艺 / 舞蹈 / 演技 / 魅力 / 气质 / 体能}
```

读取成功后，通告接洽标题下方的属性总览栏会显示实时数值。
